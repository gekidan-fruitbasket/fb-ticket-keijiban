# Change Log

All notable changes to this project will be documented in this file.

## [v1.2.4] - 2025-12-12

### Reverted
- **ヘッダーレイアウト切り戻し**: 左寄せレイアウト(v1.2.3)を、ロゴ・タイトル中央揃え(v1.2.2)の状態に戻しました

## [v1.2.3] - 2025-12-12

### Changed
- **ヘッダーレイアウト変更**: ロゴを左端、タイトルを中央、インフォアイコンを右端に配置するレイアウトに変更

## [v1.2.2] - 2025-12-12

### Fixed
- **規約文言の統一**: 利用ガイドページの文言を、登録モーダル内の最新版（簡潔な表現）に統一

## [v1.2.1] - 2025-12-12

### Fixed
- **ヘッダーレイアウト**: インフォアイコンの垂直位置がずれていた問題を修正（中央揃え調整）

## [v1.2.0] - 2025-12-12

### Added
- **規約同意フロー**: 初回登録時に利用規約への同意を必須化（チェックボックス追加）
- **ヘッダー導線**: 全画面共通のヘッダー右上に「利用ガイド」へのアクセスボタンを追加

## [v1.1.0] - 2025-12-12

### Added
- **利用ガイド・規約ページ**: アプリの目的、免責事項、禁止事項（転売禁止、誹謗中傷禁止など）、運営方針を明記したページを追加
- **ホーム画面への導線**: リスト下部に「利用ガイド・規約」へのリンクボタンを追加

## [v1.0.0] - 2025-12-12

### Released
- 初回リリース
- チケット交換機能の実装
- チケット求む機能の実装
- マイページ機能（取消、成立管理）
- LIFF + GASによるバックエンド連携

### Added
- **UI/UX**:
  - アプリライクなヘッダー（戻るボタン、タイトル）
  - フォーム下部のキャンセルボタン
  - カスタム確認モーダルの実装（ネイティブダイアログの置き換え）
- **Features**:
  - 「文化庁チケット」の選択肢追加
  - 初回登録モーダルのデザイン調整（絵文字削除）
